# 5. HTML Tag 
Tag dalam HTML seperti sebuah perintah dalam HTML, biasaya selalu terdiri dari pembuka dan penutup.


# 6. Attributes
Setiap element dalam HTML dapat memiliki attribute (element adalah struktur lengkap dari tag)

Contoh Element dalam HTML:
```html
<h1>Heading H1</h1>
```

1. id
2. class
3. href
4. src
dan masih banyak lainnya 

Contoh penggunaan Attribute
```html
<h1 id= "">Heading H1</h1>

```

# 7. List
Dalam HTML kita dapat membuat list dengan menggunakan Unordered List (ul) atau Ordered List (ol) tergantung pada jenis daftar yang ingin ditampilkan. Tag (ul) digunakan untuk membuat daftar tidak berurutan, di mana item ditandai dengan simbol seperti bullet atau lingkaran, sedangkan tag (ol) digunakan untuk membuat daftar berurutan yang ditampilkan dengan angka, huruf, atau format lainnya. Setiap item dalam daftar tersebut dibuat menggunakan tag (li) (list item). 

Contoh penggunaan list (ul):
```html
<ul>
  <li>Item Pertama</li>
  <li>Item Kedua</li>
  <li>Item Ketiga</li>
</ul>
```

Contoh penggunaan list (ol):
```html
<ol>
  <li>Langkah Pertama</li>
  <li>Langkah Kedua</li>
  <li>Langkah Ketiga</li>
</ol>
```

# 8. Link
Dalam HTML, link digunakan untuk membuat hyperlink yang memungkinkan pengguna berpindah dari satu halaman ke halaman lain, membuka dokumen, atau menuju ke lokasi tertentu dalam halaman yang sama. Tag (a) (anchor) digunakan untuk membuat hyperlink, dengan attribute href (Hypertext Reference) yang menentukan URL atau tujuan link. Link dapat diterapkan pada teks, gambar, atau elemen lain, sehingga memberikan navigasi yang dinamis dalam sebuah website. Misalnya, untuk membuat link teks

Contoh Membuat link teks:
```html
<a href="https://www.instagram.com/nickolaswewee/">Info IG</a>
```

Contoh Membuat link gambar: 
```html
<a href="https://www.example.com">
  <img src="gambar.jpg" alt="Gambar Link">
</a>
```

# 9. Image
Image pada html ditampilkan dengan menggunakan Tag (img) yang merupakan sebuah void element, sehingga dalam tag img ini tidak memiliki tag penutup 

Contohnya: 
```html
  <img src="" alt="">
```

dalam tag (img) memerlukan sebuah atribut penting untuk fungisnya yaitu atribut src dan alt. atribut src digunakan untuk menentukan lokasi gambar yang akan ditampilkan, yang bisa berupa absolute URL (https://example.com/image.jpg) atau relative URL (images/photo.jpg) tergantung pada lokasi file gambar. dan atribut alt digunakan untuk memberikan deskripsi alternatif yang akan ditampilkan jika gambar tidak dapat dimuat, misalnya karena kesalahan file atau koneksi internet.

Contohnya: 
```html
    <img src="images/photo.jpg" alt="Foto pemandangan pegunungan yang indah">
```

# 10. Div
Tag div dalam HTML adalah elemen dengan sifat display block yang digunakan sebagai container atau wadah untuk mengelompokkan elemen-elemen HTML lainnya. (div) sering digunakan untuk memisahkan dan mengorganisasi bagian-bagian dalam sebuah halaman web, seperti menu, konten utama, footer, header, atau sidebar, sehingga memudahkan pengaturan tata letak dan styling menggunakan CSS. Elemen (div) tidak memiliki makna semantik bawaan, sehingga biasanya dipadukan dengan atribut seperti class atau id untuk memberikan identitas atau peran tertentu pada bagian tersebut.

Contoh penggunaan Div: 
```html
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Contoh Div</title>
  <style>
    .header {
      background-color: #4CAF50;
      color: white;
      text-align: center;
      padding: 10px;
    }
    .content {
      padding: 20px;
      background-color: #f9f9f9;
    }
    .footer {
      background-color: #ddd;
      text-align: center;
      padding: 10px;
    }
  </style>
</head>
<body>
  <div class="header">
    <h1>Header</h1>
  </div>
  <div class="content">
    <p>Ini adalah konten utama halaman.</p>
  </div>
  <div class="footer">
    <p>Footer Â© 2024</p>
  </div>
</body>
</html>
```

# 11. Span
Tag span dalam HTML adalah elemen dengan sifat display inline yang digunakan untuk membungkus sebagian kecil teks atau elemen lain tanpa memengaruhi tata letak blok di sekitarnya. Tag ini berguna ketika Anda ingin menerapkan gaya atau manipulasi tertentu pada bagian kecil dalam teks atau elemen inline lainnya. Berbeda dengan div yang bersifat block, span tidak memulai baris baru dan memungkinkan pengaturan gaya hanya pada elemen atau teks yang spesifik.

Contohnya:
```html
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Contoh Span</title>
  <style>
    .word1 {
      color: red;
    }
    .word2 {
      color: green;
    }
    .word3 {
      color: blue;
    }
  </style>
</head>
<body>
  <p>
    <span class="word1">GDGOC</span> 
    <span class="word2">KEREN</span> 
    <span class="word3">BANGET</span>
  </p>
</body>
</html>
```

pada contoh diatas  tag span digunakan untuk membungkus setiap kata dan menerapkan kelas CSS yang memberikan warna berbeda pada setiap kata tanpa memengaruhi tata letak paragraf secara keseluruhan sehingga tag span ini sangat cocok untuk penggunaan di mana elemen inline perlu diubah atau dihias secara spesifik.

# 13. Comment 
Komentar digunakan untuk menambahkan keterangan atau deskripsi pada kode untuk developer, komentar tidak akan ditampilkan

Contoh Comment pada Html:
```html
<!-- ini adalah comment-->
```

Contoh penggunaan comment pada file HTML:
```html  
<body>
  <!-- Ini adalah komentar untuk menjelaskan struktur halaman -->
  
  <h1>Judul Halaman</h1> <!-- Judul utama halaman -->
  
  <!-- Bagian ini menampilkan paragraf pembuka -->
  <p>Selamat datang di website kami!</p>
  
  <!-- Kode di bawah ini untuk menu navigasi -->
  <nav>
    <ul>
      <li><a href="#home">Home</a></li>
      <li><a href="#about">About</a></li>
      <li><a href="#contact">Contact</a></li>
    </ul>
  </nav>
  
  <!-- Kode berikut dinonaktifkan sementara -->
  <!--
  <p>Ini adalah paragraf yang dinonaktifkan.</p>
  -->

</body>
```
